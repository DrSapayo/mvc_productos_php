# MVC Productos

Proyecto para prácticas de PHP usando el patrón MVC (Modelo Vista Controlador)

## Tecnologías usadas

- PHP
- MySQL
- PDO

## ¿Qué hace este proyecto?

- Creación de objetos Producto mediante un modelo establecido y usando prácticas como encapsulamiento.
- Conexión con base de datos en MySQL por medio de PDO para protegernos frente a inyecciones SQL.
- Gestión de errores evitando que el programa colapse aplicando buenas prácticas.

## Estructura MVC

- /mvc_productos_db
    - /app
        - /config
        - /controllers
        - /models
        - /views
    - /database
    - /public
